Machine Learning en cuatro ideas

El laboratorio imita el ciclo básico de un proyecto de clasificación de imágenes.

🖼️

Dato

Cada imagen es un ejemplo que contiene características visuales.

🏷️

Etiqueta

“Gato” y “perro” son las respuestas que tú asignas al entrenar.

⚙️

Entrenamiento

El modelo compara los ejemplos y registra patrones que separan las clases.

🔮

Predicción

Para una imagen nueva, el modelo propone la clase más parecida a lo aprendido.

Importante: esta app usa un modelo educativo de vecinos cercanos con rasgos visuales simplificados —orejas, hocico, bigotes y forma de la cara—. Una aplicación real suele aprender miles de rasgos directamente de muchos píxeles y necesita una colección mucho mayor de imágenes.
