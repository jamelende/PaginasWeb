Q-learning update

Q(s,a) ← Q(s,a) + α[r + γ max Q(s′,a′) − Q(s,a)]

The value of an action improves when it leads to immediate reward or to a promising future state.

How does the agent learn?

It alternates exploration (random actions) and exploitation (the best-known action). Epsilon slowly decreases as experience grows.

Connection to the Hugging Face course

The original Huggy activity uses Unity ML-Agents and PPO. This browser lab keeps the same agent–environment–reward idea, but uses transparent tabular Q-learning so every update can be inspected.
