Preparado

Guía rápida del laboratorio

Este laboratorio conserva la idea central del modelo NetLogo: duplicar la misma red para comparar cómo distintas medidas cambian la importancia visual de cada nodo.

  1. Seleccione una red y pulse Cargar red.
  2. En cada vista elija una centralidad y pulse Aplicar.
  3. Use el menú Layout para las 4 vistas para aplicar Resorte, Circular, Aleatorio, Cuadrícula o Radial de forma global; los botones de cada vista siguen funcionando de manera independiente.
  4. Pulse Destacar Top en todas o el botón Top de una vista para resaltar en amarillo los nodos con mayor valor.
  5. Use la rueda del mouse para hacer zoom y arrastre el fondo para desplazar la vista. Al pasar sobre un nodo se muestran sus métricas, componente y vecinos; el doble clic abre el detalle completo.
