Preparado

Guía rápida del laboratorio

Este laboratorio conserva la idea central del modelo NetLogo: duplicar la misma red para comparar cómo distintas medidas cambian la importancia visual de cada nodo.

  1. Seleccione una red y pulse Cargar red.
  2. En cada vista elija una centralidad y pulse Aplicar.
  3. Use Resorte o Circular para cambiar solamente el layout de esa vista.
  4. Active el Top para destacar en amarillo los nodos con mayor valor.
  5. Arrastre nodos y haga doble clic para ver todas sus métricas.
