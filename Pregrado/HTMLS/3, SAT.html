Historia y contexto

SAT fue el primer problema demostrado NP-completo. 3-SAT conserva la dificultad aun cuando cada cláusula tiene tres literales y se usa como punto de partida para muchas reducciones.

Pregunta de decisión: ¿Existe una asignación de verdad que satisfaga todas las cláusulas?

Verificación vs construcción

Verificación fácil: El certificado es una asignación de verdadero/falso para las variables. Verificar consiste en evaluar cada cláusula.
Construcción difícil: Con n variables hay 2ⁿ asignaciones posibles. La dificultad es encontrar una que satisfaga todas las cláusulas.
