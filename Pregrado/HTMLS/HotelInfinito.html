Pregunta central

¿Cómo puede un hotel lleno recibir más huéspedes si tiene infinitas habitaciones?

n → n+1 n → 2n pares/impares

Contexto histórico

El hotel infinito de Hilbert ilustra propiedades sorprendentes del infinito numerable. El hotel tiene habitaciones 1, 2, 3, ... y todas están ocupadas. Aun así, puede recibir nuevos huéspedes si se reindexan las habitaciones.

Objetivo didáctico: romper la intuición finita. En infinito numerable, una parte puede tener el mismo tamaño que el todo.

Reglas del juego

  1. Hay habitaciones numeradas por naturales.
  2. Inicialmente todas están ocupadas.
  3. No se expulsa a nadie.
  4. Se usa una función para mover huéspedes.
huésped antiguo n → nueva habitación f(n)

Ideas clave

Infinito numerable

Un conjunto es numerable si sus elementos pueden listarse como 1, 2, 3, ...

Parte igual al todo

Los pares son una parte de los naturales, pero n → 2n los empareja con todos los naturales.

No es contradicción

Es contraintuitivo, pero no inconsistente.

Bijección

Una correspondencia uno a uno muestra igualdad de cardinalidad.

Ventana finita

La app muestra algunas habitaciones; la prueba vive en la regla general.

Contraste

En un hotel finito lleno no hay espacio. En uno infinito numerable sí hay reindexación.
